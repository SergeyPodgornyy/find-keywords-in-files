DESCRIPTION DU POSTE  :
Etudier la possibilité d’utiliser une BDD NoSQL dans le composant « Enregistreur d’événements »
Le composant « Enregistreur d'événements » qui est en charge de tracer toutes les erreurs dans nos
produits de vidéosurveillance à besoin d’intégrer des évolutions et à besoin d’améliorer sa gestion
de sauvegarde.
Cet enregistreur d’événements est un élément critique par le fait que le fichier sauvegardé doit être
dimensionné de façon sécuritaire (il doit notamment être robuste lors des pannes de courant).
Le but de ce stage est donc d’étudier la possibilité d’utiliser une base de données NoSQL pour
stocker les événements du journal d'événements et de fournir une solution technique pour tirer parti
de certains des avantages que proposent des solutions comme «Redis » (ajustement automatique de
la taille).
Activités :
• Etudier les technologies NoSQL / redis
• Etudier la faisabilité technique
• Développer la nouvelle version de l’enregistreur d’événements
• Tester / Valider
• Évaluer l'efficacité d’utiliser une base de données NoSQL.

DESCRIPTION  DU  POSTE  :
Mission
Au sein de l’équipe technique d’un projet de développement d’envergure nationale, vous
participerez au tuning de plates-formes web (Linux / Apache / Tomcat / JBOSS / VM Java) et à leur
adaptation selon les besoins des applications Java JEE qu’elles doivent accueillir. Cela suppose de
travailler en relation avec les équipes de développement.
Il ne s’agit pas seulement d’installer et d’optimiser les composants Open Source Apache, Tomcat,
JBossen environnement complexe, mais de travailler sur l’automatisation de l’installation et du
déploiement des modules ainsi optimisés (mise en œuvre usine à RPM Koji).
L’optimisation de la configuration de la VM Java peut aussi être un axe d’optimisation du
fonctionnement de la plate-forme cible.
Technologie :




Linux, Apache, Tomcat, JBOSS, Shell, Perl, Ant, Koji, XML, ANT, VM Java, Heartbeat
Objectifs pédagogiques :
* Intégrer une équipe technique de type R&D d’un projet national
* Appréhender les contraintes en environnements web complexes
* Travailler sur l’automatisation des déploiements
* Mettre en pratique et perfectionner vos connaissances des modules Open Source Apache, Tomcat,
JBOSS
* Maitriser une plate-forme LAMP

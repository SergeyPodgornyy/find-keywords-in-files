VEILLE SUR LES FRAMEWORKS
Entreprise
Entreprise  de  maîtrise  d’œuvre  de                                                                                     200  collaborateurs,  le  GIE  SESAM-Vitale  est  au  cœur  des  échanges
électroniques  sécurisés  entre les  Professionnels  de  Santé et  l’Assurance  Maladie.  L’infrastructure mutualisée
SESAM-Vitale  gère  aujourd’hui  plus  d’un  milliard  de  transactions  par  an  en  relation  avec  plus  de  300 000
Professionnels de Santé.
Description de la mission
Dans le cadre de son activité, le GIE SESAM Vitale réalisent des applications Web.
Le GIE SESAM-Vitale souhaite réaliser une étude sur les nouveaux Frameworks JavaScript pour les applications
Web (AngularJS,  backbone.js…).  Les  Frameworks  identifiés  seront  prototypés  dans  le  cadre  d’un  projet  de
référence.
Une étude sur les outils de développement avec ces Frameworks sera également effectuée.
Description détaillée
   Développement de prototypes d’application Web
   Synthèse documentaire
Profil recherché
Etudiant(e) en 2ème année d'école d'ingénieur en informatique (conception/développement).
Pré-requis
  HTML5
  CSS3
Compétences visées durant le stage
  Frameworks
  JavaScript
  REST
  JPA
Durée / Date
3 à 5 mois / Début du stage au premier semestre 2015.

Description du poste :
Caractéristiques du poste
Dans le cadre d'un projet WEB pour l'un de nos clients, nous avons effectué une étude afin de concevoir
l'application  souhaitée  sur  la  base  des  technologies  exigées,  à  savoir                                       :  JEE,  JSF,  base  de  données
relationnelles. L'objectif du stage est de proposer une alternative à la solution issue de notre étude. C'est à
dire dans un premier temps d'adapter celle-ci sur une base NoSQL avec une interface en AngularJS, et dans
un second temps de concevoir un module d'exploitation des données basé sur le moteur de recherche
ElasticSearch.
A ce titre, et tout en respectant le processus de développement logiciel vous serez amené à :
* Concevoir les modèles objets,
* Développer les interfaces Web en AngularJS,
* Automatiser les tests unitaires avec JUNIT,
* Manipuler les objets de base de données NoSQL en utilisant JPA,
* Utiliser et configurer le moteur de recherche ElasticSearch,




* Maitriser les concepts d'intégration continue à l'aide de l'outil MAVEN,
* Utiliser les procédés de gestion de projet Atos

Les  WebSockets  sont  un  des  points  d'évolution  apportés  par  HTML5.  Ils  permettent  de  réaliser  des
applications Web temps réel, en limitant le nombre de connexions, et les données échangées entre le
navigateur et le serveur.
Les principales missions du stage sont :
- Dresser un état de l'art des serveurs applicatifs respectant la norme JSR 356 : Java API for WebSockets
- Concevoir et réaliser un prototype d'application Web utilisant les WebSockets. Le domaine d'application est
la valorisation des données spatiales.
Les travaux devront être réalisés dans le respect des règles et des contraintes s'appliquant sur les projets :
rédaction de TU/TI, tests d'intégration, norme de codage, …

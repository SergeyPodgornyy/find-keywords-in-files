Description du poste :
Pour compléter son offre de produits de sécurité des systèmes d'information, CS propose un stage de
développement autour de son offre TrustyBox d'horodatage, archivage, signature et vérification de signature.
Le stage porte sur le développement dans l'interface Web d'administration de TrustyBox d'une fonctionnalité
permettant de gérer le cycle de vie des clés et des certificats utilisés dans la mise en œuvre des différents flux
SSL du système. TrustyBox est développé en environnement Java / J2EE avec des IHM réalisées autour de
Bootstrap et Apache Wicket.
Vous serez intégré(e) à l'équipe de R pour :
- Etudier le fonctionnement de application existante ;
- Ecrire des spécifications détaillées ;
- Concevoir les évolutions de l'application en respectant les choix d'architecture des autres produits ;
- Développer et intégrer les développements.

- Langages de développement Java J2EE,

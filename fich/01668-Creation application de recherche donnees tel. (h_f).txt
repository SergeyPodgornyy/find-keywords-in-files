Description du poste :
Le stage est réalisé au sein du pôle "Network Management" qui conçoit et développe des applications de
supervision, de diagnostic et de reporting statistiques de réseau.




Le but du stage est la réalisation d'une application permettant le stockage et la sélection de données sur les
appels téléphoniques.
Les étapes du stage sont :
- Prise de connaissance générale du domaine Network Management
-  Etude  des  fonctions  actuelles  permettant  l'établissement  de  tableaux  de  bord  statistiques  sur  les
conversations téléphoniques
- Définition de la solution technique pour le stockage et la sélection des données selon différents critères
- Développement de l'application
- Test de l'application
- Installation en plate-forme de développement Atos
- Rédaction de la documentation associée.
Environnement technologique : java j2EE requise et MongoDb, Neo4j, JEE 7, Eclipse, Sonar, Hudson, SVN,
Linux, …

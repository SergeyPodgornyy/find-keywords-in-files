Stage (H/F)   - Développeur PHP/C
Projet:
Développement d’un moteur de batch processing permettant la transformation
de PDF en ePub 3 basés sur du texte
CloudAuthoring est notre outil de batch processing  pour nos clients éditeurs. Il permet
notamment de transformer des PDF en ePub. Les ePub actuels sont basés sur de l’image
signifiant que les textes sont aplatis.
Objectifs du stage:
Modification  du  moteur  de  batch  processing  actuel  pour  le  rendre  multi  grafcets.
Actuellement un seul grafcet est exécuté. L’idée est donc de proposer à l’utilisateur
de choisir son grafcet de traitement.
Récupération des informations HTML générés pour les exploiter dans la génération d’ePub
Fixed Layout maintenant le texte en tant que texte.
Récupération des mêmes informations HTML pour les exploiter dans un algorithme existant,
mais néanmoins à modifier, afin de générer un ePub Reflow Layout.

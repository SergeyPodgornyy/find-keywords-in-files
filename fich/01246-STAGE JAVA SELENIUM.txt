Mission
Dans le cadre de l’automatisation des tests fonctionnels, vous aurez à intervenir sur  une
technologie basée sur Selenium.
Les scénarios sont enregistrés avec Selenium, généré en Java par une application maison, et
lancé via Jenkins sur l’ensemble de ses plateformes de tests.
Le stagiaire aura pour mission d’enregistrer des tests fonctionnelles manuels en tests
automatisés, d’externaliser les données à paramétrer du scénario dans des fichiers CSV, de les
rejouer pour assurer leur bon fonctionnement, et enfin de les transformer en java.
Le stagiaire devra s’occuper de :
    La prise en compte des besoins sur différents projets
    La compréhension des scénarios manuels existants
    Le chiffrage des taches d’automatisation
    La réalisation de tests automatisés avec Selenium
    Paramétrage externalisés de ces scénarios et génération en Java de ces derniers
Profil
Nous recherchons un étudiant effectuant un stage de 6 mois à partir d’avril ou mai ayant
préalablement déjà effectué la réalisation d’un site Web.
Durée du stage de fin d’études : 6 mois de fin mars- début avril à fin Septembre.
Lieu : Nantes Centre
La connaissance et l’installation d’un Jenkins/Hudson serait un plus.
Compétences techniques :
Navigateur WEB, http/https
Langage Java et Javascript
Utilisation de Selenium ou autre Robot WEB (QTP, IMaccros, etc)

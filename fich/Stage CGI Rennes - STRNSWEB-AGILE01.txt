ST/RNS/WEB-AGILE/01
OBJECTIF DU STAGE
Intégrer un projet qui couvre les évolutions des applications de quotation et
de  portail  web  mis  à  disposition  par  notre  client  pour  ses  clients
internationaux.
Vous intégrez une équipe de 4 personnes dans les locaux de notre agence
de Rennes.
Applications WEB :
    Serveur d’applications : Tomcat, JBOSS, JONAS
    Technologies J2E (Framework : Spring ; Hibernate ), jBpm, dRools
    OS : Redhat Entreprise Linux, SGBD : MySQL
    Navigateurs : IE, Firefox
    Méthodologie : Scrum
MISSIONS
    Apprendre et mettre en pratique la méthodologie Agile Scrum
    Comprendre et participer à la rédaction des analyses techniques
et/ou fonctionnelles
    Développer dans un environnement technique Java J2EE, dans le
respect des normes de développement et des métriques de qualité
de code
    Rédiger des plans de tests et participer aux campagnes de tests
    Vous  bénéficierez  d’une  formation  en  interne  liée  au  contexte
spécifique (technique et fonctionnel)
PROFIL
    Formation BAC+2 ou plus
    Connaissance des technologies Web   (HTML, java, …)
    Connaissance  des  bases  de  données  (par  exemple  Oracle  ou
mysql et le langage SQL)
    Bonne pratique de l’anglais
    Bon relationnel

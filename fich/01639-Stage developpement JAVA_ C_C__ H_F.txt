Description du poste :
Il s'agit dans un premier temps de modéliser le problème, puis évaluer les possibilités et performances de
logiciels type solveurs de contraintes (tels que Gecode, Choco, eclipseclp, …) pour déterminer les séquences
d'exécution de code qui satisfont au mieux les contraintes temporelles et de bande-passante. L'utilisation de
ces outils passe par une phase préalable de modélisation du problème et des contraintes dans le langage de
l'outil de type prédicat.
Dans  un  second  temps,  le  candidat  devra  assurer  l'intégration  du  solveur  de  contraintes  dans  un
environnement représentatif d'une utilisation projet.
Environnement technologique et fonctionnel :
Java, C/C++, Shell (Python serait un plus), programmation par contraintes.

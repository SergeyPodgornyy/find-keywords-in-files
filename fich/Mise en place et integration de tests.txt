DESCRIPTION DU POSTE  :
L'objet du stage est de réaliser la mise en place de tests automatiques pour qualifier une IHM web,
mobile ou tablette. Les scripts crées devront ensuite être associés à Quality Center.
Le logiciel préconisé dans le cadre des tests est Sélénium. Ce logiciel OpenSource permet
d'automatiser les tests, et de les exporter sous différents langages (Php, Java, Python, C#…). Le
langage préconisé dans le cadre du stage est Java, potentiellement Python.
Une réflexion sur ce sujet a été entamée.
